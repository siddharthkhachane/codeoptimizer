Enter your OpenAI API key. It will only be used in your browser for API calls.
Welcome! Enter your OpenAI API key, upload a codebase (ZIP or GitHub URL), then ask questions about the code.
Note: This app runs entirely in your browser.
  • Uses OpenAI's text-embedding-3-small for embeddings and gpt-4o-mini for answering questions
  • Your API key is only used in your browser and never sent to any other server except OpenAI
  • Large repositories may take time to process and will use OpenAI API credits
  • Supported file types: .py .js .ts .java .cpp .c .go .rb .php .cs .rs .swift .kt .html .css .vue .svelte .dart .r .m .pl .lua and more
